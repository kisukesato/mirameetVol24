# 事前準備内容の確認

## GCPアカウントの確認

## Dockerの動作確認  
試しに下記のコマンドを実行すると、docker-composeのバージョンが確認できます。  
```
docker-compose version
```
![](img/docker-compose_version.png)  

## ソースファイルの取得
